Welcome

This is a simple REST API with information on VC funded companies


Data is updated roughly every month, and there are currently over 30K entries in the full dataset.
There are currently 3 test endpoints, for examples requests see the Postman documentation


# Description Endpoint Example Endpoint
1 Return 25 recently funded start-ups /funded https://vc-funded-api.herokuapp.com/funded
2 Returns companies funded by category /funded/category/{str: category} https://vc-funded-api.herokuapp.com/funded/category/Cybersecurity
3 Returns companies funded by name /funded/company/{str: company_name} https://vc-funded-api.herokuapp.com/funded/company/Tessian
